## Highlights
- 프레임워크·빌드 도구 없이 순수 HTML/CSS/JS로 구현한 다크 테마 1페이지
- 네이티브 `<dialog>` 기반 모달 뷰어 — 문서/프로젝트를 새 탭 이동 없이 그 자리에서 미리보기
- Canvas 파티클 배경, SVG 웨이브 일러스트 등 가벼운 인터랙션 디테일
